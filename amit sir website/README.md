# Photo Studio Website (Static, Multi‑Page)

This is a modern, responsive static website for a professional photo studio.

## Pages

- `index.html` — Home
- `about.html` — About
- `services.html` — Services
- `process.html` — The Process
- `portfolio.html` — Portfolio
- `contact.html` — Contact

## Customize your details

Open any `.html` file and replace the bracket placeholders like:

- `[Studio Name]`
- `[Founder Name]`
- `[Number]`
- `[Phone Number]`, `[Email Address]`, `[Physical Address]`
- `[Link to Instagram]`, `[Link to Facebook]`

## Run it

- Easiest: double‑click `index.html` to open in your browser.

If you want a local server (recommended), from this folder run:

- PowerShell:
  - `cd "C:\Users\PTJ\Desktop\amit sir website"`
  - `python -m http.server 5173`

Then open `http://localhost:5173`.

## Windows (one‑click server)

Double‑click `start-server.bat` to start the local server and open the website.
